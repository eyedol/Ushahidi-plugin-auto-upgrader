== Description ==
Auto upgrader plugin for Ushahidi to automatically upgrade Ushahidi deployments

== Installation ==
1. Copy the entire /autoupgrader/ directory into your /plugins/ directory.
2. Activate the plugin.
3. Make sure your webserver can write to the folder your Ushahidi deployment is installed in. On Ubuntu you can do sudo chmod -R www-data: [your_ushahidi_folder]
